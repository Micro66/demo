简洁高效的 Git 版本控制管理界面

常用 Git 命令

  • git init
    初始化 Git 仓库
  • git clone <url>
    克隆远程仓库到本地
  • git add .
    添加所有文件到暂存区
  • git commit -m "提交信息"
    提交暂存区的文件到本地仓库
  • git push origin main
    推送本地提交到远程仓库
  • git pull origin main
    从远程仓库拉取最新代码
  • git status
    查看工作区状态
  • git log
    查看提交历史

Git 工作流程

1

创建/克隆仓库

初始化新的 Git 仓库或克隆现有仓库

2

修改代码

在本地进行代码修改和开发

3

暂存更改

将修改的文件添加到暂存区

4

提交更改

将暂存区的文件提交到本地仓库

5

推送更改

将本地提交推送到远程仓库

6

同步更新

拉取远程仓库的最新更改
